# Pexeso Mobile Game

A premium memory game (Pexeso) built with HTML, CSS, and JavaScript. Optimized for mobile devices with a glassmorphism design.

## How to Play
1. Click 'HRÁT' to start.
2. Flip cards to find matching pairs.
3. Match all pairs in the shortest time and with the fewest moves!

## Features
- Splash screen with loader.
- Smooth animations.
- Glassmorphism UI.
- Timer and Move tracker.
